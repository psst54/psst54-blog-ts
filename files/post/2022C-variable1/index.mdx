지금까지 배운 내용을 통해서는 Hello world같은 간단한 문자열은 출력할 수 있습니다.

하지만 우리가 <Highlight>계산기 프로그램</Highlight>을 만들어야 한다고 생각해 봅시다. 사용자가 a, b 두 수를 입력하면 a + b의 값을 사용자에게 알려주고 싶습니다. 2와 3을 입력받아서 5를 출력해 주는 것처럼요. 이런 경우에 우리는 어떻게 해야 할까요?

<br />

1. 먼저, a와 b를 사용자로부터 입력받아야 합니다.
2. 이렇게 입력받은 a와 b의 값을 어딘가에 저장해야 합니다.

   (a와 b를 어딘가에 저장하지 않는다면 컴퓨터는 입력받은 a와 b를 그대로 날려버리고 맙니다...)

3. a + b를 계산합니다.
4. a + b의 결과를 출력해줍니다.

대충 이런 식으로 작동하는 프로그램을 작성해야 합니다.

<br />

이 글은 2번 내용을 다룹니다. 왜냐하면 C언어에서는 a와 b를 입력받기 전에, 미리 a와 b를 저장할 저장 공간을 먼저 만들어줘야 하기 때문입니다. 1번 내용은 다음 글에서 다룹니다.

<br /># 변수 선언하기

C언어에서는 정수, 실수, 문자 등을 저장하기 위해서 <Highlight>**변수**</Highlight>를 사용합니다.

변수는 아래처럼 <u>선언</u>할 수 있습니다.

```c
int a;
```

저는 a라는 변수를 선언했습니다.

<br />

### 정수

`int`는 이전에도 본 적 있는 단어입니다. `int main(void)`에서 봤었죠? `int`는 정수를 뜻한다고 했습니다.

이렇게 선언하면 <Highlight>**정수를 저장하는 저장 공간을 만들었고, 그 저장 공간의 이름은 a**</Highlight>라는 뜻이 됩니다.

`int sgcc;`라고 하면 sgcc라는 이름을 가진 저장 공간을 만든 것입니다. 그 다음에, 이 저장 공간에 -5라는 숫자를 저장하고 싶다면, `sgcc = -5;`처럼 숫자를 <Highlight>**대입**</Highlight>해주면 됩니다.

<br />

그렇다면 정수 말고는 어떤 데이터를 저장할 수 있을까요?

<br />

### 실수

`int`는 정수를 저장할 수 있지만, 3.14같은 실수는 저장할 수 없습니다. 실수를 저장하기 위해서는 `float`, `double`이라는 이름으로 저장 공간을 선언해야 합니다. 이 둘 모두 실수를 저장할 수 있지만, 저장 방식이 약간 다릅니다. 결론적으로 말하자면 `double`이 실수를 비교적 정확하게 저장합니다.

<br />

실수 자료형으로 선언하기 위해서는

- `float b;`
- `double c;`

처럼 쓰면 됩니다.

선언하고 나서 `double c = 3.14;` 처럼 값을 넣을 수 있습니다.

<br />

### 문자

3, 3.14같은 숫자들이 아니라, h, e, l, l, o 같은 알파벳들을 저장하고 싶을 때는 문자 자료형을 이용하면 됩니다.

문자 자료형을 선언하기 위해서는

- `char d;`

처럼 써주세요.

문자 자료형에 문자를 저장할 때는, 꼭 문자를 ‘’로 감싸줘야 합니다. 컴퓨터는

- `char d = a;`

라고 하면 a를 알파벳으로써의 a를 저장하라는 뜻이 아니라, 변수 a로 인식하기 때문입니다. 컴퓨터는 a에 저장된 값을 가져와서 d에 넣어줄 것입니다.

- `char d = ‘a’;`

처럼 써야 우리가 아는 알파벳 a를 저장할 수 있습니다.

<br />

C언어에는 더 많은 자료형이 있지만, 여기에서는 이 4가지만 다루도록 하겠습니다. 다른 자료형에 대해서는 기회가 될 때 설명하겠습니다.

<br />

# 변수에 데이터 저장하기

변수에 데이터를 저장할 때는

1. 선언할 때 대입까지 끝내기
2. 선언하고 뒤에서 대입하기

두 가지 방식이 모두 가능합니다.

```c
#include <stdio.h>

int main(void) {
	int a = 10;
}
```

```c
#include <stdio.h>

int main(void) {
	int a;
	a = 10;
}
```

<br />

# 변수를 선언하는 규칙

1. 변수 이름은 알파벳과 숫자, \_(언더바)로만 이루어져야 합니다.

   ```c
   #include <stdio.h>

   int main(void) {
   	char alphabet;
   	char SGCC;
   	char sgcc; // 위 변수와는 다른 변수입니다.
   	char hello_world;
   	char student1;
   	char student2;

   	char hello! // 불가능!
   	return 0;
   }
   ```

2. 변수 이름은 숫자로 시작할 수 없습니다.

   ```c
   #include <stdio.h>

   int main(void) {
   	char 12; // 불가능!
   	return 0;
   }
   ```

3. 한 번에 변수를 여러개 선언할 수 있습니다.

   ```c
   #include <stdio.h>

   int main(void) {
   	// 선언하고 나서 대입하기
   	int first, second, third;
   	first = 1;
   	second = 2;
   	third = 3;

   	// 선언하면서 대입하기
   	int fourth = 4, fifth = 5, sixth = 6;
   	return 0;
   }
   ```
